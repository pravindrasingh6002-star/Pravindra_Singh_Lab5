Title and Theme:-
                 Prime Properties - Real Estate Listing Platform.
                 Theme - Responsive real estate website with property listings, filters and agent profiles.

Short description of sections:-
   >Header with logo and navigation (Home, Properties, Agents, Contact).
   >Hero banner with tagline and call‑to‑action button.
   >Filters sidebar for price range, bedrooms, and property type (visual only).
   >Properties section with responsive property cards (image, price, features, location) using CSS Grid.
   >Our Expert Agents section centered in the content with agent cards (photo, experience, stats).
   >Contact section with inquiry form (name, email, phone, message).
   >Footer with quick links and contact info.
Breakpoints and layout techniques:-
    >Layout: CSS Flexbox for header, footer, and some card alignment; CSS Grid for property cards and main layout columns.
    >Breakpoints: desktop default, tablet around 1024px (sidebar stacks, single main column), and mobile around 768px and 480px (single‑column layout, stacked cards, adjusted paddings and font sizes).
How to run / view
     >Open index.html directly in a browser (Chrome recommended).
     >Or, if using a local server (VS Code Live Server), start the server and open the shown URL.
     >If deployed to GitHub Pages, include the GitHub Pages link here so faculty can view the live site.
